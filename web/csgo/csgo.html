Project Summary

The objective of this project is to create a cloud-based private Counter-Strike: Global Offensive server to be able to play multiplayer matches with, and against people online. The server will keep track of player kills, and also allow users assigned to the admin group to customize their skin through an in-game menu. An automatic back-up script creates a log file showing the history of player kills over all time.

Background

Counter Strike: Global Offensive is an online First-Person Shooter that was developed for over two years, Global Offensive was released for Windows, macOS, Xbox 360, and PlayStation 3 in August 2012, and for Linux in 2014. This project aims to create a private server environment for users to be able to configure and manage of their own accord.

Project Overview

The service is a cloud-hosted dedicated server for running Counter Strike: Global Offensive. Upon completion of implementation, a public facing Google Cloud Instance will be created for use and configuration by the administrator.
